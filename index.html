أهم القوانين الرياضية

📐 قانون فيثاغورس

a² + b² = c²

في أي مثلث قائم الزاوية، مربع وتر المثلث (الضلع الأطول) يساوي مجموع مربعي الضلعين الآخرين.

مثال: إذا كان a = 3، b = 4، فإن:
3² + 4² = c² → 9 + 16 = 25 → c = 5

📊 مساحة الدائرة

A = π × r²

مساحة الدائرة تساوي π (باي) مضروباً في نصف القطر تربيع. π ≈ 3.14159

مثال: إذا كان نصف القطر r = 5:
A = π × 5² = 25π ≈ 78.54

∫ قانون التكامل الأساسي

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C

تكامل x أس n يساوي x أس (n+1) مقسوماً على (n+1) زائد ثابت التكامل C، حيث n ≠ -1.

مثال: ∫ x³ dx = x⁴/4 + C

📈 المتوسط الحسابي

x̄ = (Σxᵢ) / n

المتوسط الحسابي يساوي مجموع القيم مقسوماً على عدد القيم. رمز Σ يعني "مجموع".

مثال: للقيم 5, 10, 15:
x̄ = (5 + 10 + 15) / 3 = 30 / 3 = 10

📐 قانون الجيب

a / sin(A) = b / sin(B) = c / sin(C) = 2R

في أي مثلث، نسبة كل ضلع إلى جيب الزاوية المقابلة له تساوي قطر الدائرة المحيطة بالمثلث.

مثال: إذا كان A = 30°، a = 5:
5 / sin(30°) = 5 / 0.5 = 10 = 2R → R = 5

🔢 محدد المصفوفة 2×2

|a b| = ad - bc
|c d|

محدد المصفوفة 2×2 يساوي حاصل ضرب العناصر على القطر الرئيسي ناقص حاصل ضرب العناصر على القطر الثانوي.

مثال: |2 3| = (2×5) - (3×1) = 10 - 3 = 7
